## Install
Clone main project
```
git clone https://github.com/messi9999/Race-Car-Tracking.git
```

### Create python environment
```
cd Race-Car-Tracking
```
```
python -m venv env
```
```
cd env/Scripts/activate
```
```
cd ..
cd ..
```
### Install requirements

```
pip install -r requirements.txt
```

## Run

```
python main.py
```
